\section*{Problem 11.16.4.9}
\begin{bf}9.\end{bf} If 4-digit numbers greater than 5,000 are randomly formed from the digits 0,1,3,5, and 7, what is the probability of forming a number divisible by 5 when
\begin{enumerate}
\item the digits are repeated?
\item the repetition of digits is not allowed?
\end{enumerate}
\section*{Solution:}
Let X be a random variable denoting four digit number formed from the digits 0,1,3,5,7.

\begin{enumerate}
\item{digits are repeated} 

In this case, X can take values 500 values between 1000 to 7777.
\begin{align} 
\pr{X} = \frac{1}{500}
\end{align}
Probability required: 
\begin{align}
\pr{(X\ mod\ 5 = 0)|(X>5000)}
\end{align}

There are 4 digits in the number, number of numbers greater than 5000 is given by:
\begin{align}
2\times 5\times 5\times 5 - 1 = 249'
\end{align}
\begin{align}
\pr{X>5000} = \frac{249}{500}
\end{align}
Of these, numbers which are divisible by 5 are given by:
\begin{align}
2\times 5\times 5\times 2 - 1 = 99
\end{align}
\begin{align}
\pr{(X\ mod\ 5)(X>5000)} = \frac{99}{500}
\end{align}
\begin{align}
Now, \pr{A|B} = \frac{\pr{AB}}{\pr{B}}
\end{align}
\begin{align}
\pr{X\ mod\ 5=0|X>5000} = 
\end{align}
\begin{align}
\frac{\pr{(X\ mod\ 5=0)(X>5000)}}{\pr{X>5000}}
\end{align}
Hence required probability is:
\begin{align}
\frac{99}{249} = \frac{33}{83}
\end{align} 
\item{digits are not repeated} 

In this case, X takes 96 different values between 1035 and 7531.
Number of numbers greater than 5000 is given by:
\begin{align}
4\times 4\times 3\times 2 = 96
\end{align}
\begin{align}
\pr{X} = \frac{1}{96}
\end{align}
Number of numbers greater than 5000 is given by:
\begin{align}
2\times 4\times 3\times 2 = 48
\end{align}
\begin{align}
\pr{X>5000} = \frac{48}{96}
\end{align}
To find \pr{(X\ mod\ 5)(X>5000)}:
\begin{enumerate}
\item{Case 1}

When leading digit is 5, number of possible numbers is:
\begin{align}
1\times 3\times 2\times 1 = 6
\end{align}

\item{Case 2}

When leading digit is 7, number of possible numbers is:
\begin{align}
1\times 3\times 2\times 2 = 12
\end{align}
Thus, total number of numbers are 12+6 = 18.
\begin{align}
\pr{(X\ mod\ 5)(X>5000)} = 
\end{align}
\begin{align}
\frac{18}{96}
\end{align}
Thus \pr{(X\ mod\ 5)|(X>5000)} = 
\begin{align}
\frac{\pr{(X\ mod\ 5=0)(X>5000)}}{\pr{X>5000}}
\end{align}
\begin{align}
= \frac{18}{48} = \frac{3}{8}
\end{align}
\end{enumerate}
\end{enumerate}
